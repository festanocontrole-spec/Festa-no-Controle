# Festa no Controle — ajustes mobile-first e vídeos YouTube

Este pacote contém os arquivos ajustados para:

1. Tornar a landing mais mobile-first.
2. Corrigir o cabeçalho no celular.
3. Incluir menu mobile e barra de tópicos/âncoras da landing.
4. Reordenar a landing para começar com promessa, dor resolvida e CTA.
5. Reescrever textos para comunicação direta com o lead.
6. Preparar a seção de vídeos com embed do YouTube via variáveis de ambiente.
7. Documentar roteiros, prompts e passo a passo para geração/publicação dos vídeos.

Arquivos principais:

- src/app/page.tsx
- src/components/AppHeader.tsx
- src/components/LogoFestaNoControle.tsx
- .env.example
- docs/comercial/landing-mobile-first-textos.md
- docs/comercial/roteiros-videos-youtube.md
- docs/implantacao/passo-a-passo-mobile-youtube.md
- scripts/zip/gerar-zip-ajustes-mobile-youtube.ps1

Observação: este pacote não inclui .env.local, node_modules, .next ou .git.

# Checklist — Quando oferecer gratuito, simbólico ou pago

## Recomendação principal

Usar a **Opção B — gratuito apenas no módulo essencial** como oferta padrão para clientes fundadores realmente estratégicos.

As opções A, C e D podem ser usadas em situações específicas, principalmente quando o lead não fecha agora, mas tem alto potencial futuro.

## Critérios para oferecer Opção B — módulo essencial gratuito

Ofereça quando a maioria das respostas for "sim":

- O evento tem data próxima e chance real de uso.
- Existe dor clara de fila, caixa, pedidos ou retrabalho.
- A organização aceita começar pequeno.
- Há uma pessoa responsável por coordenar o piloto.
- O cliente aceita fazer simulação antes do evento.
- O cliente autoriza uso como case ou aprendizado.
- O cliente tem boa rede de influência para indicação.
- O evento é parecido com o nicho que queremos vender depois.
- O escopo cabe no que o sistema já faz hoje.
- O piloto não exige customização pesada.

### Decisão

Se marcar 7 ou mais itens, pode oferecer Opção B.

## Critérios para oferecer Opção A — tudo gratuito com contrapartida forte

Use com muito cuidado. Vale apenas quando:

- o cliente é altamente estratégico;
- o case pode abrir muitas portas;
- há visibilidade institucional;
- existe autorização clara de depoimento;
- o evento tem bom volume de público;
- a equipe é organizada;
- o aprendizado será muito valioso;
- há chance concreta de contratação futura ou indicação.

### Decisão

Use Opção A apenas para 1 ou 2 cases muito especiais.

## Critérios para oferecer Opção C — taxa simbólica

Use quando o lead demonstrou interesse, mas você percebe que gratuito pode gerar pouco compromisso.

Indícios:

- o cliente quer testar, mas ainda está indeciso;
- a organização tem capacidade de pagar algum valor;
- existe risco de não levarem o piloto a sério;
- o evento não é tão estratégico como case;
- será necessário mais tempo de suporte.

### Sugestões de taxa simbólica

- R$ 197 para teste muito simples.
- R$ 297 para implantação básica.
- R$ 497 para evento com mais preparação.

### Decisão

Use Opção C quando o cliente tem potencial, mas precisa demonstrar compromisso.

## Critérios para oferecer Opção D — grátis na primeira edição, pago na próxima

Use quando:

- o cliente tem evento sazonal;
- a decisão de compra agora está difícil;
- a organização precisa provar internamente o valor;
- há chance forte de recorrência anual;
- a coordenação quer testar antes de apresentar para diretoria.

### Como posicionar

"Nesta edição, entramos como piloto fundador no módulo essencial. Se fizer sentido para vocês, a próxima edição já entra como plano comercial."

### Decisão

Use Opção D para leads bons, mas com baixa maturidade de compra no momento.

## Quando não oferecer gratuito

Não ofereça gratuidade quando:

- o cliente quer customização antes de testar o básico;
- não existe responsável definido;
- o evento está muito em cima da hora;
- o cliente não aceita fazer simulação;
- o cliente não permite feedback/case;
- há expectativa de suporte ilimitado;
- o escopo depende de funcionalidades que ainda não existem;
- a organização quer apenas economizar, sem compromisso;
- o risco reputacional é alto.

## Matriz rápida de decisão

| Situação do lead | Oferta recomendada |
|---|---|
| Alto potencial, evento real, aceita case e escopo simples | Opção B |
| Case excepcional, muita visibilidade e alta influência | Opção A |
| Interesse bom, mas precisa criar compromisso | Opção C |
| Lead bom, mas sem verba ou sem decisão agora | Opção D |
| Sem responsável, sem teste, quer customização pesada | Não oferecer gratuito |

## Oferta para lead que não virar cliente agora

Sim, vale usar A, C ou D como estratégia de recuperação, mas com cuidado.

### Sequência recomendada

1. Comece com a oferta padrão: **Opção B — módulo essencial gratuito para cliente fundador**.
2. Se o lead gostar, mas não puder decidir agora, ofereça **Opção D — primeira edição piloto, próxima edição comercial**.
3. Se o lead tiver capacidade financeira, mas estiver inseguro, ofereça **Opção C — taxa simbólica**.
4. Use **Opção A — tudo gratuito** apenas se o lead for um case de alto impacto.

## Perguntas para decidir a oferta

Antes de prometer gratuidade, responda:

1. Esse cliente pode virar case real?
2. Ele tem evento com data definida?
3. A dor é clara e urgente?
4. O escopo cabe no módulo essencial?
5. Existe responsável pelo piloto?
6. O cliente aceita simulação antes do evento?
7. O cliente aceita dar feedback?
8. O cliente pode indicar outros clientes?
9. O cliente conseguiria pagar no futuro?
10. O piloto ajuda a vender o Festa no Controle para outros?

## Regra de ouro

Gratuidade só faz sentido quando compra aprendizado, prova social ou porta de entrada comercial. Gratuidade sem contrapartida vira custo e desvaloriza a solução.
